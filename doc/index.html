Functions

set_token (tk)
Set the access token for your Dropbox account. Call this function first before you do any other calls.

Parameters:

  • tk string Dropbox access token
list_folder (path, recursive, include_media_info, include_deleted)
List contents of a folder.

Parameters:

  • path string Folder path, use empty string for ROOT level
  • recursive bool
  • include_media_info bool
  • include_deleted bool

Returns:

  1. Raw response, pass it to cursor() to iterate and paginate over the results
  2. Error message if error occured during operation otherwise you'll get only the first few hundred results

See also:

    cursor
create_folder (path)
Create a folder.

Parameters:

  • path string Folder path and name, e.g. "/Homework/NewFolder"

Returns:

  1. Raw response
  2. Error message if error occured during operation
cursor (res)
Iterate over a folder listing.

Parameters:

  • res raw response from list_folder

Returns:

    Iterable, can be used in for loops
upload (path, data)
Upload (create) a file. Should only be used for smaller files.

Parameters:

  • path string Path of file
  • data string Raw contents of file, can be text or binary

Returns:

  1. Raw response, contains meta info about created file
  2. Error message if error occured during operation
copy (from_path, to_path)
Copy a file.

Parameters:

  • from_path string Source file path
  • to_path string Destination file path

Returns:

  1. Raw response
  2. Error message if error occured during operation
delete (path)
Delete a file or folder. Will delete all the contents in a folder along with it.

Parameters:

  • path string Folder or file path, e.g. "/Homework/Folder"

Returns:

  1. Raw response
  2. Error message if error occured during operation
